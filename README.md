#NichanUrlParser

##概要
にちゃんねる互換BBSのsubject.txtやdatの取得を外部クラスライブラリ化するためのテストアプリ兼クラスライブラリのサンプルアプリ。
後日、クラスライブラリをDLLとして配布予定


##趣旨
* したらばドメイン移転で発生した各種ツールでの読み込み不全を解消することを目標に、以下の機能実装を予定しています。
  - データ取得部分のクラスライブラリ化
  - 取得データのURL、subject.txt、datファイル定義の外部XML化
  - Webアクセス部分の非同期化
